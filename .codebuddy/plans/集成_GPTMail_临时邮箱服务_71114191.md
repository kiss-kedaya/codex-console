## 功能概述

GPTMail 是一个免费临时邮箱服务，具有以下特点：

- API 地址：https://mail.chatgpt.org.uk
- 无需 API Key，自动生成邮箱
- 使用 JWT token 进行认证
- 支持 Chrome impersonate 模拟浏览器指纹
- 内置域名黑名单机制（记录不支持 OpenAI 注册的域名）

## 核心功能

1. 创建临时邮箱 - 访问首页获取 browser token，调用 generate-email API
2. 获取邮件列表 - 通过 x-inbox-token 认证获取邮件
3. 提取验证码 - 从邮件内容中提取 6 位验证码
4. 域名黑名单管理 - 自动记录不支持 OpenAI 的邮箱域名

## 技术栈

- 语言：Python 3.10+
- HTTP 客户端：curl_cffi（支持浏览器指纹模拟）
- 架构模式：继承 BaseEmailService 抽象基类，通过 EmailServiceFactory 注册

## 实现方案

### 服务架构

采用与现有 TempmailService 相同的架构模式：

1. 继承 `BaseEmailService` 基类
2. 实现必需的抽象方法：`create_email`、`get_verification_code`、`list_emails`、`delete_email`、`check_health`
3. 通过 `EmailServiceFactory.register()` 注册服务

### 认证流程

1. GET `/` 访问首页，提取 `window.__BROWSER_AUTH.token`
2. 使用 `x-inbox-token` header 进行 API 认证
3. Token 是 JWT 格式，payload 包含 email 信息

### 域名黑名单

- 内存缓存 + 文件持久化
- 自动检测 "unsupported"、"blocked"、"account_deactivated" 等错误
- 黑名单文件：`data/gptmail_banned_domains.txt`
